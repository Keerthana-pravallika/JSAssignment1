Console Methods :

console.clear() method - Used to clear the console

console.log() method - To display the text in the console

console.error() method - Used to log error message to the console

console.warn() method - Used to log warning message to the console

console.table() method - Used to display contents in form of table.

console.count() method - Used to count the number

console.timeEnd() and console.time() - Used to know the time taken to run the code

console.info - Outputs an informational message to the console
